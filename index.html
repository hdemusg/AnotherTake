Overview and Demo

Written in Python, AnotherTake is an application that allows users to send a chat bot, Twilio, clips of audio and put effects onto it. To demo it you must be added to the verified list. See the demo section above for more information.

How to use it. It's Simple.

Start by texting the Twilio bot, and upload your audio for thr Twilio bot to use. After you select the desired effects the Twilio bot will return the effects to you to download.

How it works. It's amazing.

Once the phone number receives a memo it is passed to a GCP Cloud Function, which then returns an audio of the effects selected for the user to download.
